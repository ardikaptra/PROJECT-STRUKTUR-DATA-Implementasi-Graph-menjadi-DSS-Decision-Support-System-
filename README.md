# DSS Pemilihan Beach Club Terbaik di Bali

## Deskripsi

Sistem Pendukung Keputusan (DSS) untuk memilih Beach Club terbaik di Bali menggunakan:

- Graph
- Weighted Graph
- Adjacency List
- Priority Queue
- Algoritma Dijkstra

## Struktur Project

dataset.csv
dijkstra.py
path.py
main.py

## Cara Menjalankan

```bash
python main.py
```

## Output

Menampilkan:

- Ranking Beach Club
- Jalur Terbaik
- Cost Minimum

## Algoritma

Dijkstra

Kompleksitas:

O((V+E) log V)
